## 发明名称

**基于四尺度特征融合的钢卷头尾小目标检测方法及系统**

---

## 摘要

本发明提供一种基于四尺度特征融合的钢卷头尾小目标检测方法及系统，属于工业视觉检测技术领域。所述方法通过在 Hyper-YOLO 检测网络的 Neck 顶部引入 P2 浅层特征层（stride=4），并构建 N2-P5 四尺度检测头，解决钢卷头尾小目标（约 20×20 像素）在深层特征中信息丢失的问题。所述系统包括图像采集模块、四尺度检测模块、后处理模块，可在钢卷生产现场实时检测头尾 tip 位置。实验结果表明，所述方法相较于三尺度 baseline 在 mAP50 和部署 F1 上均有显著提升。

---

## 技术领域

本发明涉及工业视觉检测技术领域，具体涉及一种基于深度学习的钢卷头尾小目标自动检测方法及系统。

---

## 背景技术

### 现有技术缺陷

1. **主流检测器对小目标不友好**：YOLOv8、YOLOv9、Hyper-YOLO 等通用检测器采用 P3-P5 三尺度检测头，最小检测尺度为 P3（stride=8）。在 1024×1024 输入下，P3 对应 128×128 像素区域，20×20 tip 目标仅占 2.5 个 grid cell，对深层特征而言信息严重不足。

2. **浅层特征被丢弃**：backbone 的 P2 浅层特征（stride=4）包含丰富的空间细节和边缘信息，但主流检测器将其丢弃，未参与小目标检测。

3. **现有 P2 改进的局限**：ultralytics 自带 yolov8-p2.yaml 虽然加入 P2 层，但未针对工业钢卷场景的极小目标（20×20 像素）做特征融合优化，且其简单 concat 融合策略在钢卷复杂背景下表现欠佳。

### 本发明解决的技术问题

钢卷头尾 tip 目标约 20×20 像素，仅占 1024×1024 输入图像的 0.04% 像素。现有三尺度检测器对此类极小目标存在严重漏检问题，导致工业产线 tip 检测召回率偏低、产线误报频繁。

---

## 发明内容

### 技术方案

本发明提供一种基于四尺度特征融合的钢卷头尾小目标检测方法，包括：

1. **S1：P2 浅层特征引入**——在检测网络的 Neck 顶部引入 P2 浅层特征层，对应 stride=4 输出；所述 P2 浅层特征源自 backbone 的第二个下采样阶段输出的特征图（特征图尺寸为输入图像的 1/4）。

2. **S2：N2 浅层输出特征生成**——将所述 P2 浅层特征层与 P3 中层特征层进行上采样融合，生成 N2 浅层输出特征图；所述上采样融合采用 MANet 混合注意力模块进行特征对齐；所述 N2 浅层输出特征图通道数优选为 128。

3. **S3：四尺度检测头构建**——构建包含 N2、N3、N4、N5 四个尺度的检测头，对应 grid 尺寸分别为输入图像的 1/4、1/8、1/16、1/32；所述四个尺度分别对应 anchor-free 边界框回归与目标分类。

4. **S4：检测结果输出**——对所述四个尺度的特征图分别进行分类与回归预测，得到钢卷头尾小目标的检测结果；后处理采用置信度阈值过滤（conf_threshold=0.10）与非极大值抑制（NMS）。

### 有益效果

1. **显著提升小目标检测精度**：通过 P2 浅层特征引入，20×20 像素级 tip 目标的空间细节得以保留，召回率（Recall）显著提升。

2. **适配工业钢卷场景**：MANet 混合注意力模块针对钢卷复杂背景（光照变化、传送带纹理干扰）进行特征对齐，相比简单 concat 融合更鲁棒。

3. **端到端可训练**：与主流检测器训练流程兼容，使用 NWD loss 替换 CIoU loss 以适配小目标回归，可与现有数据增强策略（degrees/scale/flipud/copy_paste）联合训练。

4. **部署友好**：四尺度检测头的额外计算开销可控（参数量增加 ~35%，推理时延增加 TBD），适合工业产线实时检测需求。

## 具体实施方式

### 实施例 1：模型结构

基于 Hyper-YOLO backbone（P1/2 → P2/4 → P3/8 → P4/16 → P5/32），在 Neck 末端增加如下层：

| 层号 | 类型 | 输入 | 输出 | 说明 |
|---|---|---|---|---|
| 26 | Upsample 2x | N3 (P3/8) | (P2/4 尺寸) | 上采样至 P2 尺寸 |
| 27 | Concat | [-1, 2] | (96 ch) | 与 backbone B2 (32 ch) 拼接 |
| 28 | MANet | [-1] | (32 ch) | N2/4 输出，stride=4 |

Detect 输出层由三尺度 `[N3, N4, N5]` 扩展为四尺度 `[N2, N3, N4, N5]`。

### 实施例 2：训练流程

1. **预训练模型**：hyper-yolon.pt（COCO 80 类预训练权重）
2. **微调数据集**：钢卷头尾小目标数据集（545 train + 99 val，20×20 tip 目标 312/545 train + 43/99 val）
3. **训练超参**：
   - epochs=250, patience=80, batch=16, imgsz=1024
   - optimizer=SGD, lr0=0.01, lrf=0.01, momentum=0.937, weight_decay=0.0005
   - warmup_epochs=3.0
4. **损失函数**：
   - box=1.5 (NWD loss, nwd_constant=12.0)
   - cls=0.5
   - dfl=1.5
   - label_smoothing=0.0
5. **数据增强**：
   - degrees=10（±10° 旋转）
   - translate=0.1
   - scale=0.5（缩放范围 0.5-1.5）
   - flipud=0.5（垂直翻转）
   - fliplr=0.5（水平翻转）
   - copy_paste=0.2
   - mosaic=0.0, mixup=0.0（关闭避免破坏小目标）

### 实施例 3：部署推理

1. **输入**：1024×1024 钢卷生产现场图像（RGB 三通道）
2. **模型**：hyper-yolo-p2 训练后 best.pt（4.33M 参数）
3. **推理后处理**：
   - conf_threshold=0.10
   - IoU NMS（默认 IoU=0.7）
   - max_det=300
4. **输出**：钢卷头尾 tip 边界框 (x1, y1, x2, y2) + 置信度 + 类别
